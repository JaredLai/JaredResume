Little Green House English Academy Website

A job that was originally tutoring kids in their ESL classes quickly turned into a web development project once my employers learned about my major. I created a website from the ground up, bought a domain, and designed the content. Facebook Pixel was also integrated to allow Facebook Ads to be posted, and the website was also configured to allow integration with Google Ads.

What I learned:
This was my first time doing learning about websites and HTML/CSS. Although we were using Wix as a easy way to deploy and design our website, the user interface was quite unpleasant. I quickly discovered manually coding what I envisioned proved to be much more efficient. The auto-generated code lacked comments and adequate variable naming, making it quite challenging to decipher. Counterintuitively, using Wix forced me to learn much more about coding a website than I imagined. This project helped me quickly familiarize myself with web development technologies. Working with HTML code generated by Wix forced me into deeper understanding of the HTML language.

Shutterbug

Modeled drone with SolidWorks and fabricated via 3D-printing and laser-cutting. The goal was to code an Arduino to drop payloads upon detecting certain objects, a specific piece of colored paper in our case. On the hardware side of things, we soldered a PixyMon V2, servos, motors, and power supply. These components were then wired to the Arduino and a DJI flight controller.

Our main challenge:
The scope of this project spans across mechanical, electrical, and software engineering. We had to make sure our motor and propeller pair was powerful enough to generated enough lift for our quadcoper, while balancing overall weight and power consumption to fit the specifications our battery allowed. The frame of the quadcoper had to be structurally sound but lightweight. It also had to house all our wiring and components. Soldering and wiring components proved to be taxing as well, as most of us were learning this for the first time. Having all different aspects of the quadacopter work together seamlessly was our main challenge. Despite the many setbacks, I was able to experience the intricacies of completing a complicated project and learned to excel as a closely knit team.
